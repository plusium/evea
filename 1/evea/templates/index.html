简易使用方法：

1：注册一个账号，登录后增加一个角色。

2：在角色列表页面获得角色的 keyID 和 vCode。

3：使用一个独立的最新版 EVEMon，在设置界面（General --> Network）添加一个 API Provider，地址为 http://evea.sinaapp.com

4：在 EVEMon 设置界面的 General --> Updates 里禁用时钟同步和自动更新，将 Account Status 以下的所有 Query 的时间间隔改为 Never。

5：使用第 2 步得到的 keyID 和 vCode 在 EVEMon 中添加角色（File --> Add API Key）。

6：以后随时登录网站，使用角色管理界面或者快速技能导入工具，对角色属性信息及技能信息进行更新。

详细使用方法见ECF帖子
